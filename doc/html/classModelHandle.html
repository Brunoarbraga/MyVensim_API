void ModelHandle::printModel ( )
inlinevirtual

printing function for the Model class.

Function that prints on terminal all the model informations, that is, all the flows and its atributes, all the systems and its atributes, and the Model name.

Can be used to verify if the Model is looking as expected throughout the simulation, if needed.
