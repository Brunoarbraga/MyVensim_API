\doxysubsection{Detailed Description}
Logistical flow class derived from flow. 

Represents a logistical interaction beetween systems, flow is sort of inverted, taking the value from the destination \doxylink{classSystem}{System} and transfering it to the origin \doxylink{classSystem}{System}. Its only purpose is to implement the pure virutal method \doxylink{classLogisticalFlow_aa380b703376d79abc5de059b18540e41}{run()}, from flow.
